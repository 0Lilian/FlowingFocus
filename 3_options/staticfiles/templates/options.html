Appearances > Header

If the header is sticky it will be fixed on the top of your page content even if you scroll down.

If this setting is 'on' and if you have choosen the 'Ultra-condensed' header's layout, the layout will be switched to 'Condensed' each time you hover the header.

If checked, views' hotkeys will be displayed each time your mouse hover a view button.

If checked, actions' hotkeys will be displayed each time your mouse hover a action button.
